Remote Android Developer

Location: Kolkata

Type: Full Time

Job Description

We're seeking a passionate and skilled Android Developer to join our mobile team. You'll be responsible for the design, development, and maintenance of high-performance, user-centric Android applications for our clients. The ideal candidate has a strong understanding of the Android lifecycle, design patterns, and a commitment to writing clean, efficient code. Key Responsibilities

  • Develop and maintain robust, native Android applications using Kotlin and/or Java.
  • Collaborate closely with product managers, UX/UI designers, and backend engineers to define and implement new features.
  • Ensure the performance, quality, and responsiveness of applications.
  • Identify and correct bottlenecks, fix bugs, and improve application stability and performance.
  • Work with external data sources and APIs (RESTful, GraphQL).
  • Unit-test code for robustness, including edge cases, usability, and general reliability.
  • Stay up-to-date with new mobile technology trends, application architectures, and Android platform updates.


Required Skills & Experience:

  • Education: Bachelor's degree in Computer Science, Engineering, or a related field (or equivalent practical experience).
  • Experience: X+ years of professional experience in native Android development. (Replace X with a specific number, e.g., 3+)
  • Language Proficiency: Strong proficiency in Kotlin (preferred) and/or Java.
  • Android SDK: Deep understanding of the full mobile development lifecycle and the Android SDK, including knowledge of various Android versions and screen compatibility.
  • Architecture: Experience with modern Android architectural components (e.g., MVVM, MVI) and dependency injection frameworks (e.g., Dagger, Hilt, Koin).
  • Version Control: Proficiency with Git and standard development workflows (e.g., pull requests, code reviews).
  • Testing: Experience writing unit and integration tests (e.g., JUnit, Mockito, Espresso).
